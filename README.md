# LukasNicoTankstelle

Zur Funktion:
Der Benutzer unserer Tankstelle kann an drei Zapfhähnen tanken. Des weiteren kann er von drei Benzinarten wählen. Beim drücken auf den Button mit "Tanken"
wird der Tankprozess gestartet und beim erneuten drücken wird er wieder gestartet. Solange man nicht den Fertig Button drücket kann man auch noch weitertanken.
Beim betätigen des Fertig Buttons wird diese Zapfsäule gesperrt und kann erst nach dem Zahlen wieder gebraucht werden.

Zum erweitern:
Die einzelnen Komponenten im Frontend wurden als UserControls implementiert, d. h.es wäre sehr leicht noch mehr von diesen zu Instanzieren.
